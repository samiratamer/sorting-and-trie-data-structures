# Sorting and Trie Data Structures

Two C programs implementing fundamental data structures and algorithms: heap sort and a Trie for dictionary word storage and retrieval.

---

## Part 1: Heap Sort

Sorts an array of integers using the heap sort algorithm.

### Sample Output
Original array: 12 11 13 5 6 7
Sorted array:   5 6 7 11 12 13

### Concepts Used
- Max-heap construction
- In-place sorting
- Recursive heapify
- Time complexity: O(n log n)

---

## Part 2: Trie – Dictionary Word Insertion & Search

Creates a Trie data structure and implements insertion and search operations for dictionary words.

### Sample Output
Search for "hello": Found
Search for "world": Found
Search for "trie": Found
Search for "data": Not Found
Search for "algo": Not Found
Search for "algorithm": Found

### Concepts Used
- Trie (prefix tree) data structure
- Dynamic memory allocation with malloc
- String traversal and character indexing
- Boolean end-of-word markers
